## 1. nob-sakuma.com -- 佐久間宣行事務所（Grand Prix受賞）

### 1-1. ファーストビューの設計思想

- **見せているもの**: 最新作品（デスキスゲーム/Netflix）のビジュアルと、佐久間宣行のポートレート写真。「好きが渋滞しているくらいで、ちょうどいい」というキャッチコピー
- **隠しているもの**: ナビゲーション、会社概要、連絡先。「何屋か」の説明を意図的に省略し、人物の世界観で引き込む
- **設計思想**: 企業サイトではなく「人間のブランドサイト」。コンテンツへの没入を最優先し、UIの存在感を極限まで消している。スクロールを促す「SCROLL」表示あり
- **特筆すべき点**: ハッシュタグ風の番組名がマーキー（横流れ）で表示される演出。TVプロデューサーという職業を直感的に表現

### 1-2. 色彩設計

| 役割 | HEXコード | 特徴 |
|------|-----------|------|
| プライマリ | `#2B3240` | ダークネイビー。重厚感と知性を両立 |
| アクセント | `#C2F542` | ネオンライムグリーン。ダーク背景に映える蛍光色 |
| 背景 | `#FFFFFF` | 白背景ベース |
| テキスト | `#202020` | ほぼ純黒。可読性最優先 |

- **色彩の独自性**: ダークネイビー×ネオンライムという異色の組み合わせ。エンタメ業界のエネルギーとクリエイターの知性を同時に表現。一般的な法人サイトの「安心の青+白」パターンを完全に避けている

### 1-3. タイポグラフィ

| フォント | 役割 | 特徴 |
|----------|------|------|
| 秀英角ゴシック銀 B JIS2004 | 見出し | 日本語プレミアムフォント。角ゴシックながら上品な曲線 |
| Inter | 本文・英字 | モダンなサンセリフ。可読性重視 |
| Noto Sans JP | 日本語本文 | 汎用性のバックアップ |
| Montserrat | 装飾英字 | セクション見出しに使用 |
| PT Mono | コード・特殊表記 | 等幅フォントでアクセント |

- **h2サイズ: 76.384px** -- 非常に大きい見出しで視覚的インパクトを確保
- **本文: 12px** -- 小さめの本文で「情報量」より「空間」を重視
- **文字間/行間**: ゆったりとした行間設定。テキストが「呼吸している」印象

### 1-4. レイアウトの独自性

- **非対称レイアウト**: 左右非対称の写真配置。人物写真とテキストが斜めにオーバーラップ
- **散らし配置**: 番組ロゴやサムネイルが不規則に散りばめられている。まるで机の上に資料を広げたような「作業場」感
- **水平スクロール**: 作品一覧で横スクロールを採用。グリッドを完全に排除
- **タブ型のインタラクション**: 過去48作品をTab操作で切り替える。リスト表示を避けカード型に集約

### 1-5. 余白の設計

- **セクション間余白: 不均一**。コンテンツの感情的な重要度で余白量を変えている
- **「MY POLICY」セクション前後に大きな余白** -- 最重要メッセージであることを余白で暗示
- **写真周りの余白は意図的に狭い** -- オーバーラップを多用し、要素同士の「会話」を生む

### 1-6. アニメーション/トランジション

- **ハッシュタグマーキー**: 番組名が無限横スクロールで流れる
- **スクロール連動**: セクション遷移時のフェードイン/パララックス
- **GIFアニメーション**: 「面白い」のテキストがアニメーションGIFで動く
- **ホバーエフェクト**: 作品カードのホバーで詳細情報がスライドイン
- **ローディング**: 「読み込まれました」表示あり -- カスタムローダー

### 1-7. 写真/グラフィック/イラストの使い方

- **人物写真を軸に構成**: 佐久間宣行本人の写真が随所に登場。「個人」を主役にする設計
- **SVGアイコン**: 番組ジャンル（バラエティ、ラジオ等）をカスタムSVGアイコンで分類
- **書籍カバー画像**: 作品としてフルサイズ表示。Amazon連携で購買動線も確保
- **帯文グランプリ**: 書籍の帯コメントを視覚的に再現するセクション -- ユニークなコンテンツ設計

### 1-8. テンプレ感を排除している具体的な技法

1. **職業そのものをデザインに変換**: TVプロデューサーの「番組表」「クレジットロール」の文法をWeb上に再構成
2. **セクション名がエンタメ用語**: ON AIR、BOOKS、MY POLICYなど、一般的な「サービス」「実績」を使わない
3. **テキストに口語体を採用**: 「まあ見てやってください」「長いんすよ（笑）」
4. **作品一覧のUI**: 一般的なグリッドではなく、Tab + フォーカスで詳細が切り替わるカルーセル風
5. **帯文グランプリ**: 書籍の帯コメントという独自コンテンツでセクションを構成
6. **エンベッドサンドボックス**: 外部サイト（STUDIO Embed）を積極的に組み込み、コンテンツの立体感を演出
7. **ボタンのborder-radius: 0px**: 角張ったボタンで、丸みを帯びたテンプレ感を排除

### 1-9. コピーライティングの特徴

- **キャッチコピー**: 「好きが渋滞しているくらいで、ちょうどいい」 -- 詩的で比喩的。本人の言葉遣い
- **自己紹介文**: 一人称「俺」、口語体。「条件は2つ。俺の力が使えそうか。そして、わけわかんなそうかどうか。」
- **番組紹介**: 各番組の紹介文が「企画書のプレゼン」のような熱量を持つ
- **全体トーン**: 敬語を使わず、読者と対等な関係を構築。「見てやってください」

## 2. fujiwara-i.com -- 藤原印刷（法人・印刷会社）

### 2-1. ファーストビューの設計思想

- **見せているもの**: 「わたしたちの実績」として制作した書籍のスライドショー。具体的な作品を先に見せることで「何ができるか」を即座に伝える
- **隠しているもの**: 会社概要、沿革、設備紹介。よくある「創業○年」的な情報は後方に配置
- **設計思想**: 印刷会社でありながら「本をつくる喜び」という感情面を前面に出す。製造業の機能訴求ではなく、クリエイターとの共創を訴求

### 2-2. 色彩設計

| 役割 | HEXコード | 特徴 |
|------|-----------|------|
| プライマリ | `#F76700` | 鮮やかなオレンジ。情熱・温かみを表現 |
| アクセント | `#57B300` | グリーン。自然・成長を連想 |
| 背景 | `#F9F9F9` | オフホワイト。純白を避け柔らかさを演出 |
| テキスト | `#333333` | ダークグレー。純黒を避けた優しい印象 |

- **色彩の独自性**: オレンジ×グリーンという「補色に近い」大胆な組み合わせ。印刷会社らしいインク的な鮮やかさと、本の紙を連想させるオフホワイト背景

### 2-3. タイポグラフィ

| フォント | 役割 | 特徴 |
|----------|------|------|
| 見出ゴMB1 | 見出し | モリサワの見出し用ゴシック。力強くも柔らかい |
| Inter | 英字本文 | モダンサンセリフ |
| ヒラギノ角ゴシック / Hiragino Sans | 日本語本文 | Apple環境での美しい描画 |
| Noto Sans JP | フォールバック | クロスプラットフォーム対応 |

- **h2: 28px / 本文: 18px** -- 適度な文字サイズで可読性を担保
- **見出しフォント「見出ゴMB1」**: 印刷・組版の専門会社ならではのフォント選択。自社の専門性をタイポグラフィで証明

### 2-4. レイアウトの独自性

- **書籍的なレイアウト**: 印刷物の「ページ」を意識した構成。横書きでありながら縦の余白が雑誌的
- **カード型と横並びの混在**: 「コンシェルジュ」「コンサルティング」を横並びで対比表示
- **矢印（→）を用いたCTAリンク**: 統一されたリンクスタイルが設計の一貫性を生む
- **ボタンのborder-radius: 0px**: 角張った矩形ボタン。印刷物の「裁断面」を連想させる

### 2-5. 余白の設計

- **セクション間: 中程度の均一余白** -- 書籍の「章」間のような一定のリズム
- **borderRadius: 4px** -- わずかな丸みを加えたカードで、硬すぎず柔らかすぎない印象
- **フッターの情報設計**: 「奥付」というリンク名。書籍の最終ページ用語をWeb上で使用

### 2-6. アニメーション/トランジション

- **実績スライドショー**: 書籍作品の自動送りカルーセル（一時停止・前後ナビあり）
- **シンプルなページ遷移**: 過度なアニメーションを避け「印刷物のように落ち着いた」佇まい
- **ローディング**: カスタムローダー表示

### 2-7. 写真/グラフィック/イラストの使い方

- **書籍のカバー写真を大胆に使用**: 実績を「作品ギャラリー」として提示
- **SVGロゴ**: メインロゴとサブロゴ（「教えて！藤原印刷」等）をSVGで精緻に管理
- **OG画像**: ブランドの世界観を反映した専用画像

### 2-8. テンプレ感を排除している具体的な技法

1. **ナビゲーション用語の独自化**: 「奥付」「心刷祭」「クラフトプレス」など業界特有の言葉を使用
2. **サービスカテゴリの人間的な分け方**: 「個人・クリエイターのみなさまへ」「企業・チームのみなさまへ」と対象者別に語りかける
3. **「教えて！藤原印刷」**: FAQ的コンテンツに親しみやすい名前をつけている
4. **「みんなから見た藤原印刷」**: 第三者視点のコンテンツセクション。自社PRではなく他者の声を構造化
5. **フォント選択が自社の専門性を証明**: 見出ゴMB1というプロ用フォントの採用が、組版会社としての矜持を示す
6. **コーポレートカラーがインク色を連想させる**: オレンジが「版画用インク」的な暖かみを持つ

### 2-9. コピーライティングの特徴

- **タグライン**: 「つくるよろこびをつくる」 -- 動詞の反復による力強さ
- **OG description**: 「戦後ひとりの女性タイピストがはじめた印刷会社」 -- 歴史をドラマチックに要約
- **セクションコピー**: 「本の作り手は多様化し、つくりたい本もそれぞれ十人十色です」 -- 多様性を肯定する包容力

## 3. kinzokuen.co.jp -- 株式会社金属園（製造業）

### 3-1. ファーストビューの設計思想

- **見せているもの**: GIFアニメーションによる金属素材のビジュアル。テキストよりもビジュアルの質感で「レアメタル」という特殊素材の存在感を訴求
- **隠しているもの**: 具体的な取引条件、価格情報。製造業でありながら工場写真を前面に出さず、素材そのものの美しさにフォーカス
- **設計思想**: レアメタルという「地味な素材」をデザインの力で魅力的に見せる逆転の発想。製造業サイトの常識（工場・設備・安心安全アピール）を覆している

### 3-2. 色彩設計

| 役割 | HEXコード | 特徴 |
|------|-----------|------|
| プライマリ | `#FFEA00` | ビビッドイエロー。金属の輝きを象徴 |
| アクセント | `#0281DB` | ブルー。工業的な信頼性 |
| 背景 | `#FFFFFF` | 白。クリーンな印象 |
| テキスト | `#0281DB` | ブルーをテキスト色にも使用。統一感 |

- **色彩の独自性**: ゴールド（イエロー）×インダストリアルブルー。金属＝ゴールドの連想を直球で活用しつつ、ブルーで知的さを担保。テキスト色にブルーを使う大胆さ

### 3-3. タイポグラフィ

| フォント | 役割 | 特徴 |
|----------|------|------|
| Syncopate | 見出し（英字） | 幅広の大文字サンセリフ。工業的で力強い |
| Zen Kaku Gothic New | 本文（日本語） | モダンな角ゴシック。Googleフォントで無料利用可 |

- **本文: 12px** -- コンパクトなサイズ。情報密度より空間を重視
- **Syncopateフォント**: 字間が広く、金属の「精密さ」を表現。産業用書体としての説得力

### 3-4. レイアウトの独自性

- **一文字ずつの縦書き分解**: 「リサイクル」「品質管理」「持続可能」などのキーワードを一文字ずつ縦に並べて強調。日本語の縦書き文化をモダンに再解釈
- **GIFアニメーションのオーバーラップ**: 複数のGIFが重なり合い、金属加工のプロセスを暗示
- **SVGグラフィックの多用**: 装飾的なSVGを背景に配置し、工業的なパターンを形成
- **マーキー（横流れ）**: 取扱品目を横スクロールで表示。一覧の退屈さを回避
- **ボタンのborder-radius: 200px**: 完全に丸い「ピル型」ボタン。工業的な硬さとソフトなUIの対比

### 3-5. 余白の設計

- **セクション間: 大きく不均一** -- 各セクションが「展示室」のように独立
- **画像と文字の余白を意図的に狭くし、GIF群は密に配置** -- 素材の密度感を演出
- **セクション間には大きな白場** -- 「息継ぎ」としての余白

### 3-6. アニメーション/トランジション

- **GIFアニメーション多用**: 金属の質感変化、加工プロセス、選別作業をGIFで動的に表現
- **マーキースクロール**: 取扱品目のテキストバンドが無限ループ
- **サイドバーGIFアニメ**: 画面右端に縦長のGIFアニメーションが常駐。ページトップへのナビゲーション
- **ローディング**: カスタムローダー

### 3-7. 写真/グラフィック/イラストの使い方

- **GIFアニメーションが主役**: 静止画ではなくGIFで金属素材の質感変化・輝き・加工プロセスを表現
- **webP形式の高品質写真**: 「希少金属と持続可能なリサイクル」「品質管理が重要なレアメタル」のセクションで大判写真
- **SVGによる装飾グラフィック**: 抽象的なSVG図形がレイアウトにリズムを与える
- **LINEのQRコード画像**: 問い合わせ動線に生活ツールを組み込む

### 3-8. テンプレ感を排除している具体的な技法

1. **GIFアニメーションの戦略的使用**: 製造業の「静的な硬さ」を動きで打破。素材そのものがコンテンツになる
2. **一文字ずつの分解表示**: 「リサイクル」→「リ/サ/イ/ク/ル」。視覚的リズムを生成
3. **英語の大胆使用**: 「Items We Handle」「Contact Us about RARE METAL」で国際的な印象
4. **カラーパレットが業種を超越**: 製造業＝グレー・ブルーの常識を破り、ゴールド×ブルーの鮮やかさ
5. **マーキーテキストバンド**: フッター付近のブランドロゴの無限スクロール
6. **LINE問い合わせの導入**: B2B製造業にカジュアルな連絡手段を設置する意外性
7. **アクセシビリティ考慮**: フォーカスに関する表記（Tab操作の説明）がある

### 3-9. コピーライティングの特徴

- **テーマの格上げ**: レアメタルを「限られた資源」と表現し、「人類の未来に貢献」と社会的意義を付与
- **キーワードの視覚化**: 「人類/未来/貢献」を太字で強調し、メッセージを三段構成に
- **対比構造**: 「希少」「持続可能」という矛盾を対比させ、企業の存在意義を浮き彫りに

### 4-6. アニメーション/トランジション
