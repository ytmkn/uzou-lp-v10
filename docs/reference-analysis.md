## BtoB/法人LPでテンプレ感を回避するための共通パターン

10サイトの分析から抽出した、テンプレ感を避けるための実践的パターンを以下にまとめます。

### カラー・ビジュアル系

1. **背景色を純白(#FFFFFF)にしない**: ウォームグレージュ(#E6E1DE)、クリームホワイト(#FFFCF7)、アイスブルー(#F5FBFB)、ウォームベージュ(#F2EFE7)など、わずかに色味を持たせた背景色を使う。10サイト中6サイトが白以外の背景色を採用している

2. **テキストカラーに純黒(#000000)を使わない**: ダークブラウン(#4A200C)、ダークグレー(#494B4D, #333333)などにすることで、画面全体の硬さを和らげる。IMAのようにブランドカラー(#DB1527)をテキストに使う大胆な手法もある

3. **ブランドカラーを1色に絞り、全面的にコミットする**: IMA（レッドのみ）、StartPass（ライムグリーン+ブラック）のように、使う色を極端に絞ることで記憶に残るビジュアルアイデンティティを構築する

### タイポグラフィ系

4. **有償・プレミアムフォントを使う**: Tazugane Gothic、こぶりなゴシック、ゴシックMB101、リュウミン、Helvetica Now Display、Tapir Pro Bold等。Google Fontsだけでは出せない品質感がテンプレ感を排除する。特に和文フォントの選定が差別化の鍵

5. **和文・欧文フォントの組み合わせに個性を出す**: Noto Sans JPやヒラギノの定番組み合わせを避け、Zen Kaku Gothic Antique + Montserrat、こぶりなゴシック + Poppins、IBM Plex Sans + DM Sansなど、意外性のあるペアリングを行う

6. **極端なフォントサイズの振り幅を作る**: StartPassのh2: 160px、THOのh1: 48px、TANT GRAPHICSのbody: 13pxのように、通常のスケールから逸脱したサイズを意図的に使うことで視覚的なリズムを生む

### レイアウト・構造系

7. **CTAボタンの形状・エフェクトに個性を持たせる**: KAGAMIの50px blur発光シャドウ、StartPassのpill型+ライムグリーン、THOの角丸0px+黒ボーダー、ライデンのボタン不在戦略など。CTA周りは最も目に留まる要素であり、ここの差別化が全体の印象を決める

8. **マーキー/無限スクロール要素を取り入れる**: IMAの「NEXT IMPACT COMPANY」マーキー、StartPassのロゴマーキー。静的なページに動的な流れを加え、印刷物やテンプレートではできない表現を実現する

9. **インラインフォームやインラインコンテンツで遷移を減らす**: TANT GRAPHICSのページ内問い合わせフォーム、gmlabsのインラインinput。別ページへの遷移なしに完結することで、シームレスな体験を提供する

### コンテンツ・コピー系

10. **冒頭にマニフェスト/ポエティックコピーを配置する**: ライデン「我々は資本主義経済の中に生きています」、スペースマーケット「日本各地に、価値を眠らせたままの場所があふれています」、IMA「今この瞬間に未来をつくる力が宿っていること」。機能説明ではなく世界観で入ることで、テンプレ感の最大要因である「よくあるSaaS LP構成」から脱却する

11. **ナンバリングやラベルで情報の構造を可視化する**: StartPassの(01)(02)(03)、TANT GRAPHICSの「誌面設計」「字体設計」、THOのPROJECT 01/02。カード列挙ではなく、番号やカテゴリラベルで視覚リズムを生む

12. **具体的な数値やデータを大胆に配置する**: StartPassの「導入1,000社超」「出資内諾率44%」「CxO候補者500名」、KAGAMIの「1人で100人」。BtoB/法人サイトでは抽象的な価値提案より、具体的数値が差別化と信頼構築の両方に効く

### 技術・実装系

13. **SVGベースのカスタムグラフィック/ロゴタイプを多用する**: IMA、StartPass、TANT GRAPHICS、ライデンはSVGでカスタムロゴタイプやイラストを配置。画像ではなくSVGにすることで、解像度に依存しない鮮明さとカスタム感を両立する

14. **全サイトがStudio.Design製**: 10サイト全てがStudio.Designで構築されている。Studio.Designのフレキシブルなレイアウトシステムが、同じプラットフォームでありながら全く異なるデザイン表現を可能にしている。ツールの選定も差別化の一つ

15. **コンタクトフォームのUIにもブランド性を持たせる**: TANT GRAPHICSの「入力した内容に間違いはありません」チェックボックス+「SEND」ボタン（pill型黒）、gmlabsのインラインフォーム。フォームは最もテンプレ化しやすい要素だが、ここまでブランドを通すことでプロフェッショナリズムを証明する
