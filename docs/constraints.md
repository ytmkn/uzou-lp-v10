# 制約カード: UZOU LP v10

## 目的
「足し算バイアス」を防ぐ。v9では結晶パーティクル、3Dチルト、マグネティックボタン、テキストスプリット、マウス追従グロー等、技法を盛り込みすぎた反省を踏まえ、v10では「少ない要素を精密に使う」方針。

---

## 数量制約

| 要素 | 上限 | 根拠 |
|---|---|---|
| **アニメーション種別** | 最大 **5** 種類 | 回路の信号には種類が少ない。同じ動きの反復が「規則性」を生む |
| **装飾色** | アクセント含め **7** 色以内 | ブランドカラー5色 + パルスアンバー + 回路グレー。これ以上は不要 |
| **装飾パターン** | 最大 **3** 種類 | 回路パターン / パルスグロー / グラデーション遷移帯。これで十分 |
| **フォントファミリー** | 最大 **2** 種類 | Noto Sans JP + Space Grotesk。混ぜすぎると回路の精密さが崩れる |

---

## アニメーション5種の定義

| # | アニメーション | 用途 | 技法方針 |
|---|---|---|---|
| 1 | **フェードアップ** | 全セクションの入場アニメーション | CSS scroll-driven animations（view()）+ IO フォールバック。translateY(20px) + opacity |
| 2 | **スタッガード** | カード・リスト項目の順次表示 | transition-delay を nth-child で 80ms 刻み |
| 3 | **カウントアップ** | Scale セクションの数値 | IntersectionObserver トリガー。requestAnimationFrame で数値を増加 |
| 4 | **パルス明滅** | 回路ノードの発光、CTAボタンのグロー | CSS animation（ease-in-out、2-4秒周期）。box-shadow / opacity の変化 |
| 5 | **線描画** | Solution セクションの接続線、Flow の接続線 | SVG stroke-dashoffset をスクロール連動で 0 へ |

**禁止アニメーション（v9からの除外）**:
- テキストスプリット（文字単位分割）→ 過剰、パフォーマンスコスト
- 3Dチルト → 結晶コンセプト由来、v10では不要
- マグネティックボタン → 過剰インタラクション
- Canvas パーティクル → パフォーマンスコスト高、SVGパルスで代替
- 水平スクロール → 操作性の問題

---

## 装飾パターン3種の定義

| # | パターン | 用途 | 実装方針 |
|---|---|---|---|
| 1 | **回路パターン** | Hero/Final CTA の背景装飾 | SVG。直線の接続線 + 円形ノード（4-6px）。ノードがパルスアニメーションで明滅 |
| 2 | **パルスグロー** | CTAボタン枠、数値カード上辺、アクティブ要素 | box-shadow でティール色のグロー。hover / pulse animation |
| 3 | **グラデーション遷移帯** | セクション間の境界 | linear-gradient で背景色を滑らかに接続。高さ 80-120px |

**禁止装飾（v9からの除外）**:
- Glass-morphism（backdrop-filter: blur） → 結晶コンセプト由来
- ポリゴン/ファセット形状 → 結晶コンセプト由来
- ノイズテクスチャ → テイストが合わない
- 波形/斜め/有機ブロブのセクション区切り → テンプレ感

---

## パフォーマンス制約

| 指標 | 目標値 |
|---|---|
| LCP | 2.5秒以内 |
| CLS | 0.1以下 |
| INP | 200ms以下 |
| JS バンドルサイズ | 50KB以下（minified） |
| CSS バンドルサイズ | 30KB以下（minified） |
| WebFont | 2ファミリ以内、必要ウェイトのみ読み込み |
| 画像 | WebP + srcset、遅延読み込み（hero以外） |
| アニメーション | 60fps 維持。transform/opacity のみ使用（layout triggerを避ける） |

---

## レスポンシブ制約

| ブレークポイント | 幅 | 備考 |
|---|---|---|
| Desktop | 1024px+ | 基準レイアウト |
| Tablet | 768-1023px | 2カラム→1カラム化の判断 |
| Mobile | -767px | 1カラム、タップターゲット44px以上 |
| Small Mobile | -480px | フォントサイズ微調整 |

---

## アクセシビリティ制約

| 項目 | 要件 |
|---|---|
| コントラスト比 | WCAG AA（通常テキスト 4.5:1、大テキスト 3:1） |
| フォーカス | :focus-visible リング（ティール系） |
| reduced-motion | 全アニメーションの無効化対応 |
| セマンティクス | header/main/footer/nav/section/article 使用 |
| 構造化データ | FAQPage + Organization |
| キーボード操作 | Tab/Enter/Escape で全機能操作可能 |
| スキップリンク | メインコンテンツへのスキップリンク設置 |
