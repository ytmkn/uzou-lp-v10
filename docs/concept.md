# LPディレクション: UZOU LP v10

## ブリーフ分析

### 業種・サービス
- **サービス名**: UZOU（ウゾウ）
- **運営**: 株式会社Speee（東証スタンダード 4499）
- **種別**: ネイティブ広告配信プラットフォーム（DSP / AdTech）
- **USP**: 広告主・代理店とWebメディアを直接つなぐ独自商流基盤。AIによる配信最適化で、広告主の成果とメディアの収益を同時に最大化

### ターゲット
| 区分 | 属性 | 課題 | 意思決定要素 |
|---|---|---|---|
| プライマリ | 広告代理店メディアバイヤー（30-40代）| CPA改善、新規メディア開拓、配信先の質 | 実績数値、メディアネットワーク規模、運用サポート |
| プライマリ | 事業会社マーケ担当者 | CVR向上、広告効果の可視化、ROI説明 | AI最適化の具体性、導入の容易さ、費用対効果 |
| セカンダリ | Webメディア収益責任者 | 収益最大化、広告品質、UX維持 | 単価の高さ、広告の質、導入負荷 |

### 目的・KPI
- **目的**: 資料ダウンロード（主要CTA）/ お問い合わせ（副次CTA）のコンバージョン獲得
- **KPI**: CVR（資料DL率）、直帰率低減、スクロール率

### 競合差別化

## コンセプト

### v9との差別化方針
| 要素 | v9（Crystal Formation） | v10（Signal Pulse） |
|---|---|---|
| メタファー | 結晶の生成・成長 | 信号の脈動・伝播 |
| 世界観 | 鉱物的・静的・生成プロセス | 電気的・動的・瞬間の接続 |
| 主要モチーフ | ポリゴン、結晶、ファセット | パルス波、回路線、シグナルドット |
| 背景処理 | Canvas結晶パーティクル | SVG回路パターン + パルスアニメーション |
| カード表現 | Glass-morphism（結晶面） | 回路ボーダー + パルスグロー |
| 色の使い方 | グラデーション（結晶の面） | 単色の濃淡 + アクセントのパルス発光 |
| 全体印象 | 美しい鉱物標本 | 生きている回路基板 |

### コンセプトワード: **Signal Pulse（信号の脈動）**

> UZOUは、広告主とメディアの間に走る「信号」そのものである。
> 混沌としたアドネットワークの中で、UZOUのAIが最適な信号経路を見つけ出し、
> 広告という電気信号を正確に、瞬時に、最適な場所へ届ける。
> その一つ一つの信号の脈動が、広告主の成果とメディアの収益を同時に生み出す。

### 世界観
| 要素 | 定義 |
|---|---|
| 空間 | 精密に設計された回路基板の内部。無数の信号線が規則正しく走り、交差するノードで情報が処理される。ミニマルだが、隅々まで設計意図が行き届いた空間 |
| 時間 | パルス（脈動）の時間。0.1秒単位の瞬間と、それが累積して生まれる大きな成果。リアルタイム性と持続的な成長の二重の時間軸 |
| 質感 | マットなダーク基板の上を走る、わずかに発光する信号線。触れると微かな振動を感じるような、精密機器の質感。冷たすぎず、テクノロジーの温もりがある |
| 温度 | クール寄りの中間。無機質なテクノロジーではなく、「つながり」を生み出す有機的な温度。ティール色が持つ「冷静さの中の信頼感」 |

### デザイン原則
1. **Precision（精密）** — 全ての要素に意図がある。余白、線、角度、全てが計算された配置。回路設計のような精密さがUZOUのテクノロジーの信頼性を物語る
2. **Pulse（脈動）** — 静止画に見えて、実は全てが微かに動いている。スクロールに連動したパルスアニメーション、ホバーでの信号点灯。生きているプラットフォームであることを伝える
3. **Connection（接続）** — 孤立した要素はない。全ての要素が線やリズムで接続されている。広告主とメディアをつなぐUZOUの本質をビジュアルで表現する

## コンセプト数値制約

| 要素 | レンジ | コンセプトからの根拠 |
|---|---|---|
| セクション余白 | 80px〜160px | 回路基板の「精密な間隔」。均一ではなく、情報密度に応じて3段階（80/120/160px）に変化させる |
| アニメーション速度 | 200ms〜800ms | パルス信号の速度感。入場アニメーションは400-600ms、ホバーは200-300ms、背景パルスは2000-4000ms |
| フォントウェイト差 | 見出し700-800 vs 本文400 | 回路の「主線」と「副線」の太さの差。明確なヒエラルキー |
| 色の明度差 | 60%以上 | 回路基板のダーク面と発光信号のコントラスト。暗い背景での信号の視認性 |
| 影のblur範囲 | 0px〜24px | 信号のグロー。ぼかしすぎない精密なグロー（blur 8-16px が基本帯） |

## トーン&マナー

### ビジュアルトーン 5軸
| 軸 | ポジション | 根拠 |
|---|---|---|
| フォーマル ↔ カジュアル | フォーマル寄り 70/100 | 東証上場企業Speeeのプロダクト。代理店・マーケターへの信頼感が必須 |
| テック感 ↔ 温かみ | テック感寄り 75/100 | AI最適化・DSPというテック本質。ただし「つながり」の温度は残す |
| ミニマル ↔ リッチ | ミニマル寄り 65/100 | v9のリッチ（結晶、パーティクル過多）からの差別化。精密な抑制 |
| 権威的 ↔ 親しみ | 権威的寄り 65/100 | 500+メディア、東証上場の信頼。ただし押し付けがましくない |
| 静的 ↔ 動的 | 動的寄り 60/100 | パルスの動き。ただしv9の過剰アニメーションは避け、意味のある動きだけ |

### タイポグラフィ方針
- **和文**: Noto Sans JP（ブランド規定。400/500/700/800）
- **欧文**: Space Grotesk（Google Fonts）— 幾何学的でありながら角が独特の切り落としを持つ。回路の精密さとテック感を体現。Interより個性的で、v9との差別化にもなる
- **フォントの性格**: Space Groteskは「技術的だが無機質ではない」。ティール系カラーとの相性が良く、AdTechプラットフォームとしての精密さを表現する

### カラーディレクション
ブランドカラー（ティール系5色）は必須で使用。v9との差は「色の使い方」で出す:

| 色 | HEX | v10での使い方 |
|---|---|---|
| ライトティール | #8BC0CA | 信号のグロー、パルスの発光、ホバー時のアクセント |
| ミドルティール | #34626F | CTAボタン、主要リンク、アクティブ状態 |
| ディープティール | #2B4954 | 見出しの重要パーツ、回路線の色 |
| ダークティール | #1F353E | ダークセクション背景（v9よりさらに暗く、信号が映える） |
| ブラック | #040404 | テキスト（ライトセクション）、深層背景 |

**追加色（v10固有）**:
| 色 | HEX | 用途 |
|---|---|---|
| シグナルホワイト | #F4F8F9 | ライトセクション背景（純白ではなくティールのかすかな気配） |
| 回路グレー | #1A2830 | ダークセクションのサブ背景、回路パターンの基板色 |
| パルスアンバー | #F0A848 | 警告・課題セクションのアクセント（信号の「注意」色）|

**v9との色使いの差**:
- v9: グラデーション多用（結晶の面の反射）→ v10: 単色+グロー（信号の明滅）
- v9: 白↔ダーク交互 → v10: ライトグレー基調 + ダークは2セクションのみ（ヒーロー、Final CTA）
- v9: カラーパレットを面で使用 → v10: 線と点で使用（回路的）

### モーションディレクション
- **動きの人格**: 精密で規則的。ランダム感は排除。信号が回路を流れるような予測可能な動き
- **イージング**: `cubic-bezier(0.25, 0.46, 0.45, 0.94)` — 穏やかな加速から自然な停止。電子信号の動き
- **パルスアニメーション**: `ease-in-out` でゆっくり明滅。背景の回路パターンのノードが順番に発光
- **入場アニメーション**: `translateY(20px)` + `opacity: 0` → 元の位置。控えめだが確実な存在感

### v9との構成差
- v9: 12セクション → v10: 10セクション（凝縮して密度を上げる）
- v9: CTA Mid（中間CTA）あり → v10: なし（スクロール疲れ防止、Final CTAに集中）
- v9: About（プラットフォーム図解）→ v10: Solution（課題→解決の文脈でより説得力のある図解）
- v9: ダーク3セクション → v10: ダーク2セクション（ヒーロー + Final CTA のみ。ライト基調で回路パターンの精密さを見せる）

### 各セクションの情報設計

#### 1. Hero — 信号の覚醒
- **目的**: 3秒で「何のサービスか」「どんな価値があるか」を伝える
- **レイアウト**: 中央揃え。上段にキャッチコピー、下段にサブコピー + 2つのCTA
- **背景**: ダーク基板色 + SVG回路パターン（ノードがパルス発光）
- **特記**: v9のスプリットヒーロー（左テキスト/右画像）から中央型へ変更。メディアバイヤーが一瞬で把握できるレイアウト

#### 2. Trust — 信号の到達先
- **目的**: 「500以上のメディアが使っている」という社会的証明
- **レイアウト**: ロゴマーキー（横スクロール） + 実績サマリー（接続メディア数、累計配信数、導入企業数）
- **特記**: v9と同様だが、マーキーの速度をゆっくりに。回路の「信号が流れる」メタファーとして

#### 3. Problem — ノイズの検出
- **目的**: ターゲットが抱える課題に共感し、「解決策がある」と期待させる
- **レイアウト**: 2カラム（広告主の課題 / メディアの課題）。各3点の課題をリスト表示
- **特記**: v9のタブ切替から2カラム並列表示へ変更。両面の課題を同時に見せることで「両方を同時に解決する」UZOUの価値に自然につなげる

#### 4. Solution — 信号経路の設計図
- **目的**: UZOUがどう課題を解決するかを視覚的に説明
- **レイアウト**: 中央にSVGの接続図解（広告主 → UZOU AI → メディア）。左右にテキスト説明
- **特記**: v9のAboutを発展。静的な図ではなく、スクロール連動でノードが順に点灯する図解。UZOUの「直接接続」「AI最適化」を視覚的に理解させる

#### 5. Scale — 信号の増幅
- **目的**: 具体的な数値で信頼を裏付ける
- **レイアウト**: 4つの数値カード（横一列）。カウントアップアニメーション付き
- **数値**: 接続メディア数 500+、月間配信数 XX億imp、導入企業数 XXX社+、平均CPA改善率 XX%
- **特記**: 各数値カードの上辺にティールのパルスライン（回路ボーダー）

#### 6. Features — 信号処理の3つの核
- **目的**: UZOUの3つの主要機能を詳細に説明
- **レイアウト**: 縦積み3ブロック（交互レイアウト: 左テキスト-右ビジュアル / 右テキスト-左ビジュアル）
- **機能**: (1) AI配信最適化、(2) 独自メディアネットワーク、(3) 専任運用サポート
- **特記**: v9のベントグリッドから交互レイアウトへ変更。各機能にSVGアイコン + 回路的な装飾線

#### 7. Voices — 信号の反響
- **目的**: 導入企業の具体的な声で決断を後押し
- **レイアウト**: カードグリッド（3列）。各カードに引用、会社名/役職、成果数値
- **特記**: v9の横スクロールカルーセルからグリッドへ変更。スクロール操作不要で全件一覧

#### 8. Flow — 接続プロトコル
- **目的**: 導入の容易さを伝え、心理的ハードルを下げる
- **レイアウト**: 水平ステップ（4ステップ）。各ステップが回路のノードのように接続線でつながる
- **ステップ**: お問い合わせ → ヒアリング → アカウント開設・タグ設置 → 配信開始・最適化
- **特記**: v9のタイムライン（縦）から水平ステップへ変更。ノード接続で「簡単」を表現

#### 9. FAQ — トラブルシューティング
- **目的**: 残りの疑問を解消し、CVへの障壁を除去
- **レイアウト**: アコーディオン形式（左にカテゴリラベル、右に質問リスト）
- **特記**: v9とほぼ同じ構造だが、回路的な装飾（開閉時のパルスアニメーション）を追加

#### 10. Final CTA — 信号の発信
- **目的**: 最終コンバージョン。資料ダウンロード + お問い合わせの2択
- **レイアウト**: ダーク背景 + 中央揃え。大きなコピー + 2つのCTAボタン
- **背景**: 回路パターンの全ノードが同時に脈動する「全回路点灯」状態
- **特記**: v9のCanvasパーティクルからSVGパルスアニメーションへ。軽量かつインパクト

### 守るべきルールと自由度の範囲
**守るべきルール（逸脱不可）**:
1. ブランドカラー5色は必ず使用（追加色は可）
2. 和文 Noto Sans JP は変更不可
3. Vanilla HTML/CSS/JS のみ（CDN/フレームワーク不使用）
4. 全アニメーションに `prefers-reduced-motion` 対応
5. WCAG AA コントラスト準拠
6. v9のモチーフ（結晶、ポリゴン、ファセット、Glass-morphism）は使用禁止

**自由度のある範囲**:
1. 回路パターンの具体的なSVGデザイン
2. パルスアニメーションのタイミング・パターン
3. カード・ボタンの具体的なCSS値
4. レスポンシブブレークポイントの詳細
5. 追加の装飾色（コンセプト数値制約の範囲内）
6. 欧文フォントのウェイト配分
