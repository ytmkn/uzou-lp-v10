# コピー骨格: UZOU LP v10

## コピーライティング方針

### トーンルール
- **主語**: 「UZOU」を主語にする箇所と、「あなた（広告主/メディア）」を主語にする箇所を意図的に切り替える
- **硬さ**: 広告業界の専門用語は使うが、説明的すぎない。プロが読むことを前提
- **リズム**: 短い文を重ねる。1文20文字以内を基本。体言止め・倒置を適度に
- **v9との差**: v9は「広告が、メディアを救う。」のようなポエティック路線 → v10は「精密・具体的・機能的」路線

## キャッチコピー（Hero）

### 推奨案
**「届く広告は、つながりから生まれる。」**

- 根拠: UZOUの核心価値は「広告主とメディアの直接接続」。「つながり」は Signal Pulse コンセプト（信号の接続）と合致しつつ、温度感もある。「届く」は成果を暗示
- 文字数: 15文字。ヒーローで60-72pxで表示しても2行以内に収まる
